## 介绍

**Lime 青柠记账** 是一款开源的产品级项目，并提供了一整套的产品级项目解决方案，同时也是做个人 MVP 产品最好的参考案例。该项目基于 Flutter 3.X 进行 App 部分的开发，后端采用 支持 TypeScript 的 Nestjs 开发，接口和 ORM 用 GraphQL + Prisma 的方式进行数据交互，数据库使用 PostgreSQL 同时还提供了 Figma 产品原型以及 PowerDesigner 画的数据库模型图，甚至还有包括了产品相关的文档资料，总之没有最好，只有更好！！！

## 特性

- 基于全新的 Flutter 3.X
- 后端采用支持纯 TypeScript 开发的 Nestjs
- 使用了下一代的 ORM 框架 Prisma
- 为了追求更快的响应速度在 Web 框架底层使用了 Fastify 引擎
- 用 GraphQL 代替了古老的 REST API
- 使用了比 MySQL 更快的数据库 PostgreSQL
- 资料齐全，包括但不限于原型、物理模型、产品文档等
